# ERP Desktop Application

A comprehensive desktop ERP solution designed for business resource tracking and data management.

## 🚀 Demo Application
Don't want to compile the code? You can run the application instantly:
* **[Download the Pre-Compiled Windows App (ZIP)](https://github.com)**
* *Instruction:* Extract the downloaded ZIP file and double-click `AppBordir.exe` to run.

## 🛠️ Tech Stack
* **Language:** C#
* **Framework:** Windows Forms (WinForms)
* **Database:** SQLite
